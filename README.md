# Nik Holitsyn - Portfolio

A modern, interactive portfolio website built with React, showcasing skills, projects, and professional experience with stunning animations and 3D effects.

## Tech Stack

### Core
- **React** - UI framework
- **TypeScript** - Type safety
- **Rsbuild** - Fast build tool powered by Rspack
- **TanStack Router** - Type-safe routing

### Styling & Animation
- **Tailwind CSS v4** - Utility-first CSS framework
- **GSAP** - Professional-grade animation library
- **React Bits** - Animated React components library

### State & UI
- **Zustand** - Lightweight state management
- **React Icons** - Icon library
- **Custom Components** - Handcrafted UI components

## Features

- **Interactive HomePage** with light ray effects and gradient text animations
- **3D Skill Stack Gallery** with dome visualization
- **Dynamic Project Showcase** with advanced filtering
- **Responsive Design** optimized for all devices
- **Smooth Animations** using GSAP and custom CSS
- **Dark Theme** with gradient accents

## Setup

Install the dependencies:

```bash
npm install
```

## Get Started

Start the dev server, and the app will be available at [http://localhost:3000](http://localhost:3000).

```bash
npm run dev
```

Build the app for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Project Structure

```
src/
├── components/       # Reusable UI components
│   ├── icons/       # Custom SVG icons
│   └── ui/          # UI components (GradientText, SplitText, etc.)
├── pages/           # Page components (Home, About, Projects, SkillStack)
├── routes/          # TanStack Router configuration
└── utils/           # Utility functions and constants
```
